###🍷 Sistema de Cadastro de Vinhos

Este projeto é uma aplicação simples desenvolvida em JavaScript com o objetivo de simular o cadastro de vinhos em uma vinharia. A interação com o usuário é feita por meio de prompt() para entrada de dados e alert() e console.log() para exibição das informações.

###🚀 Funcionalidades

Cadastro de vinhos via prompt
Validação básica de dados utilizando estruturas de repetição (while)
Exibição das informações cadastradas no console e em alertas
Uso de funções para organizar e modularizar o código

###🛠️ Tecnologias Utilizadas

JavaScript (puro)
Execução em ambiente de navegador

###🧠 Conceitos Aplicados

Funções para separação de responsabilidades
Estrutura de repetição while para validação de entradas
Manipulação de dados fornecidos pelo usuário
Entrada e saída de dados com prompt(), alert() e console.log()
